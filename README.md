# ☀️ Solar Energy Generation Prediction Using Machine Learning

## 📌 Project Overview
This project predicts solar energy generation using weather parameters such as **irradiation, ambient temperature, and module temperature**.

The goal is to forecast solar power output and identify the most influential environmental factors affecting energy generation.

---

## 🚀 Problem Statement
Accurate solar power forecasting is important for:

- Energy optimization
- Grid stability
- Renewable energy planning
- Efficient power management

This project applies machine learning techniques to predict solar power output based on weather conditions.

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- XGBoost
- SHAP (Explainable AI)

---

## 📊 Exploratory Data Analysis (EDA)
Performed:
- Data cleaning
- Missing value analysis
- Correlation analysis
- Weather vs power visualizations

Key weather features:
- Irradiation
- Ambient Temperature
- Module Temperature

---

## 🤖 Machine Learning Models Used
1. Linear Regression  
2. Random Forest Regressor ⭐ (Best Model)  
3. XGBoost Regressor

---

## 📈 Model Performance

| Model | R² Score |
|--------|----------|
| Linear Regression | 0.9801 |
| Random Forest | 0.9864 |
| XGBoost | 0.9857 |

### Best Model:
✅ **Random Forest Regressor**

---

## 🔍 Explainable AI (SHAP)
SHAP explainability was used to understand:

- Feature importance
- Prediction behavior
- Model interpretability

### Key Insight:
**Irradiation was identified as the most influential factor affecting solar power generation.**
